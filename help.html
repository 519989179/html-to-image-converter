如何使用 HTML 转图片工具？

步骤 1：粘贴 HTML 内容

在输入框中粘贴你想要转换为图片的 HTML 代码。

步骤 2：预览 HTML 效果

点击 “Preview HTML” 按钮查看渲染效果。

步骤 3：导出为图片

点击 “Export as Image” 按钮即可下载 PNG 图片。

FAQ

  • Q: 是否支持 CSS 样式？
    A: 是的，支持大多数 CSS 样式。
  • Q: 是否支持外部资源？
    A: 不支持跨域资源（如外部图片）。
  • Q: 如何去除空白区域？
    A: 工具已自动裁剪空白部分，仅保留内容区域。
